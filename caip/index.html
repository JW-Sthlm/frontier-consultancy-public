Cloud and AI Platform · CAIP

Build AI-first delivery on Cloud and AI Platform.

A 60 to 90 minute partner workshop for Azure delivery teams. Walk through the lifecycle of an AI-first project, see the toolbelt your consultants get, and leave with the material to run it inside your organization.

What you get

A working delivery motion, not a webinar.

Agents and MCP-connected tools are starting to handle the parts of delivery that used to eat consultant hours. Backlog grooming, scaffolding, infrastructure, test, the admin around every project. The partners building this motion now have a real window.

This workshop walks you through the lifecycle of an AI-first project on Azure: client brief, planning, demo, build, ship. You see the actual toolbelt the field is using on Cloud and AI Platform: skills, agents, MCP servers, and the GitHub Copilot CLI front end that ties it together.

Everything is on GitHub. Decks, scenario, exercise, setup, FAQ. Clone once and keep it after.

Setup tiers · pick one before the session

Three depths. Pick the one your room needs.

The deck lands regardless of setup. Hands-on depth changes based on what your consultants have ready.

Tier 1 · Watch and discuss
No setup

Run through the deck and the scenario as a discussion. No installs. Best when you want a leadership-level conversation.

Tier 2 · Copilot CLI
10 min setup

Each consultant has GitHub Copilot CLI. Walk the exercise live, see real CLI sessions, no Azure deployment.

Tier 3 · Full agent stack
30+ min setup

Copilot CLI plus Azure CLI plus the agent and MCP servers from the starters. Build, deploy, ship inside the session.
