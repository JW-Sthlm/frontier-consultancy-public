# CAIP track: Cloud and AI Platform

Hands-on workshop for Azure / Cloud and AI Platform partners. The idea: AI-first delivery, where Copilot CLI plus the Microsoft Learn MCP do most of the consulting work and you supervise.

## Watch the deck

The HTML deck is served live: [jw-sthlm.github.io/frontier-consultancy-public/caip/presentation/](https://jw-sthlm.github.io/frontier-consultancy-public/caip/presentation/). Arrow keys to navigate, `f` for full screen.

## Run the exercise

Setup, scenario, deliverable templates: see [`exercise/`](./exercise/).

Run the five environment checks in [`BEFORE-WORKSHOP.md`](./exercise/BEFORE-WORKSHOP.md) before the session. If any check fails, the same doc points to a Codespaces fallback that gets you to a working terminal in about 90 seconds.

Prefer VS Code? See [`help-vscode.md`](./exercise/help-vscode.md). No GitHub or Copilot seat yet? See [`help-no-account.md`](./exercise/help-no-account.md).

## Scenario

You are a solution architect with 15 minutes before a call with Nordwind Aerospace, a satellite operator. You need a one-page brief grounded in real Microsoft guidance. Walk out with a deliverable you would send to a real client. Read [`exercise/README.md`](./exercise/README.md) to start.
