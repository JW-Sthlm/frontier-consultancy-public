Copilot Skill Deck Maker

Paste meeting notes, a recap, or a topic. Get a single-file HTML deck with brand-matched styling, hidden presenter notes, and browser navigation. No PowerPoint fight at 10:30 PM.

Open SKILL.mdView example deck

One file

The deck is self-contained HTML. Send it as a file or host it on a simple site.

Brand matched

Point it at a partner homepage. The skill samples colours, font signals, and tone.

Presenter ready

Add ?presenter=notes or press p. Notes stay out of the audience view.

3-step install

Open the skill file

The skill lives in one file: SKILL.md. View it on GitHub and copy the raw contents.

Save it into your Copilot skills folder

Save the file to ~/.copilot/skills/copilot-skill-deck-maker/SKILL.md. Create the folder if it isn't there yet. Copilot picks up new skills from this location.

Restart Copilot CLI and try it

Start a fresh Copilot CLI session. Then ask: "Make a deck from these notes: ..." or "Build slides about X, brand-matched to example-partner.com". The skill takes it from there.
