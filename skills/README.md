# Frontier Consultancy skills

Custom Copilot skills we hand to partner consultants. Drop one into `~/.copilot/skills/` and it loads on the next session.

These are the ones we build and give away. More formal, externally maintained skills we just link to. The ones here are ours.

## Available skills

| Skill | What it does |
|---|---|
| [copilot-skill-deck-maker](./copilot-skill-deck-maker/) | Turn notes, a meeting recap, or a topic into a single-file HTML deck. Brand-matched, presenter notes built in, no PowerPoint. |

## How to install one

Each skill folder has its own README with install steps. The short version:

1. Open the skill's `SKILL.md` on GitHub and copy the raw contents.
2. Save it to `~/.copilot/skills/<skill-name>/SKILL.md` on your machine.
3. Start a fresh Copilot CLI session. Skills load at startup.
