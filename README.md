# Frontier Consultancy

Partner workshop content. Built to leave with you.

If you've been invited to a Frontier Consultancy workshop, you're in the right place. Deck, exercise, setup, FAQ. Everything you need to walk in prepared and walk out with material you can keep using.

## Two ways to use this

**Watch the deck.** The HTML deck runs live via GitHub Pages. No clone needed. Open the link, arrow keys to navigate, `f` for full screen.

**Get the exercise material.** Setup guides, the exercise scenario, deliverable templates, FAQ. Clone or download the repo so you can work locally and keep the material after the session.

```bash
git clone https://github.com/JW-Sthlm/frontier-consultancy-public.git
```

Or download the zip from the green "Code" button.
