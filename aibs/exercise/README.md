## Track

This is the **ERP (Dynamics 365 Finance & Supply Chain)** exercise for the AIBS Frontier Consultancy workshop.

You work in **VS Code with GitHub Copilot Chat in agent mode**, using the D365 ERP MCP server and the MS Learn MCP server. You need a live Dynamics 365 Tier 2 or UDE environment for Phase 2.

## Reference material

- [`client/`](client/) - the two source documents (decision log, workshop transcripts) you will attach to Copilot
- [`implementation-ip/`](implementation-ip/) - the Copilot instructions file and the WHS skill that go into your workspace before you start

## Help

Stuck on prereqs? Open [erp-setup-overview.md](erp-setup-overview.md) and work through the six checks, or ask your facilitator.

Stuck in the exercise? Ask Copilot first. Paste the error, ask "what does this mean and how do I fix it". Using Copilot to unstick itself is the muscle you are training.
